补招日记

发表于 2020-11-21 更新于 2020-11-29 分类于 日常随笔 阅读次数： Valine：
秋招结束了，我青结~~本来这篇应该是《日常实习日记》的，但投了二十来家公司都没人回我，反倒是补招上岸，就变成《补招日记》了

2020.10.27

  投递公司：

  • 字节跳动
  • 快手
  • 滴滴杭州
  • 斗鱼
  • 网易有道
  • 欢聚
  • 小米
  • 京东

  投完才发现自己又忘记用内推了，害



2020.10.30

快手简历(淘汰)

  哎，快手真的难，三投三挂，实习简历都过不去



2020.11.1

​ 今天体测完略伤，一千米倒还好平时有练，但引体向上做完整个人都不好了，感觉全身都肌肉拉伤了，躺床上起身都难。以后还是要加强锻炼啊，尤其是现在还在学校这段日子



2020.11.3

京东简历(淘汰)

  又是一行“硕士优先”直接人就没了，但是我真的不觉得开发岗位硕士跟本科有啥区别啊，要是说校招因为没实习挂了我还能服气，为什么实习的技术岗也看学历啊，后来想想可能也有应届生找实习的Debuff加成在吧。

  感觉自己又开始眼高手低了，可是感觉要是去小厂打杂(虽然大厂也是打杂)在简历上没法添加优势。然而要去大厂，就得有学历、竞赛或者实习，可惜我全不能选手样样没有，之后试着投投好一点的小厂吧

网易简历(下架)

  想找大厂实习是真的太难了，竞争的人太多，而且也不缺人

  投递公司：

  • 百度
  • 汇量科技
  • SHEIN
  • 盛趣游戏
  • 帆软
  • 元戎启行
  • 京东金融

斗鱼、SHEIN、盛趣、京东金融、汇量简历(放置Play)

  实习僧简历这个已查看状态太恶心啦，看了又不改状态，搞得我每天都要上去看一看，没过也改成不合适呗

元戎启行简历(秒挂)

  到现在投了15个实习了，然而两天就已经挂四五个了。。。不想投了，心累了，要是没实习就专心一直学习准备到春招了

  不知道从什么时候开始，越是看面经越觉得自己不会的太多，无限の恶性循环



2020.11.4

  秋招遗漏下来的面试，岗位还变成数据开发工程师了，我Kafka、spark、hadoop可是样样不会啊，估计也是KPI练手了，随意吧，一轮游预备

  • 美团一面

    1. topK

      上次字节也是TOPK算法测试用例没通过，这次还是没过，SQL也没写正确，我真踏马是个fw啊，这面试官还硬是问了一小时，真是辛苦他了。又一次让我知道自己和大厂的距离。不过秋招也总该结束了吧，那实习电话为啥还不来啊呜呜呜┭┮﹏┭┮



2020.11.5

  今天还是没有实习电话的一天，有点想签了招银了，总感觉就是自讨苦吃，本来可以签了offer舒舒服服地躺几天，但是不懂为什么自己不想放弃，不是大佬心气还这么高，也有想过如果在一周期限内没找到实习就签了

  想了一晚上，犹豫就会败北，不管怎样都不签了，有这种想法感觉就容易让自己失去继续努力的动力，还是觉得不甘心啊，又投了几个部门直招的补招。

  • 微众银行
  • 小米互二
  • 小米手机
  • 京东零售云

  今天又是无所事事刷牛客的一天，哎，明天把之前记录的所有面经问题都看完把，然后开始看点技术书籍了



2020.11.6

  投递公司：

  • 滴滴北京
  • 东方财富
  • 顺丰
  • 小红书
  • 卡牛
  • 尚米网络
  • 广州卓动

  今天面经还是没看完，又是沉迷刷牛客的一天，怀念国庆那段时间啊，纯粹地刷题练算法、看书，学习提升自己，现在感觉被各种东西困扰，不过说到底还是自己的专注力不够，是时候把番茄钟下回来了

  没有梦想，那和做咸鱼有什么区别，罢了，不过是个打工人，谈什么梦想

当下的最优解：

  1. 补招上岸，去实习
  2. 找到实习，补招、春招上岸
  3. 补招、实习无望，硬着头皮刚春招
  4. 当条咸鱼，接了offer



2020.11.7

货拉拉一面

  1. 动态代理的底层
  2. spring事务中用trycatch捕获异常是否还会回滚

  又是秋招遗留下来的。有几个问题不会，但是一结束就忘记啥问题了，太草了



2020.11.8

  闲的发慌，面经不想看，题不想刷，看起了视频，追起了番，这季度的《黄金神威》和《咒术回战》很彳亍



2020.11.9

东方财富测评

  投的实习但是发了个校招的测评邮件，懵了

  想了想还是想补招上岸好啊，今天看了下知乎，今年的考研报名人数差不多得有420万，太离谱了，比去年直接就多了80w人，大学毕业生不过八百多万，考研报名的就占了一半，再加上保研名额，几乎就是全员读研了。到时候春招想必一定是人更多坑更少了，再加上一些大佬想春招再找毁约曲线救国，难以想象战况的惨烈了



2020.11.10

  补招它终于lei了，最近一直没消息又怠惰了两天，现在终于机会来了，补招上岸冲踏马的，说实话我也不想春招啊，补招上岸，学习、休息半个月再去实习它不香吗

小米一面

  1. 电脑CPU占100%怎么解决(JVM调优)
  2. HashMap插入节点是在头结点还是尾结点
  3. 翻转链表

  哎，继续投实习把，裂开来，有好几题没答出来，最后给了个像KPI一样的简单算法题，我看别人都是两道算法啊。。。感觉之前把太多希望都寄托于此了，现在失落感真的倍增。有这种补招上岸的想法是我不对，哎蒟蒻就不配上岸，投的实习到现在也不给个电话，实习机会再不来感觉连三个月的工期都保证不了了啊。。

  永远不要把鸡蛋放在一个篮子里 ——周迅



2020.11.11

  难受的双十一，至今未接到实习电话(早上九点有个电话，不过睡晚了没接到，打回去也没人接了)，下午学习也学不下去，然后听了一下午的菅田将晖，歌是真好听啊，感觉很少有男歌手的声音这么舒服的

  晚上拒绝了招银的Offer，又一次回归0offer的情况了，彻底断了退路，孤注一掷了。现在真的需要好好静下心来学习了，一切从零开始，锻炼自己的心态



2020.11.12

  机会又来了，开冲，不知道是互二的还是手机部的，但是接到电话真的很激动，感觉面试才是复习的驱动力啊！又想了想还是不要全把希望寄托在一处了，放宽心，心态保持好，从容应对吧

小米二面

  问了大部分是项目，加一些基础，加一个快排

  面试结束了，发了测评邮件，当晚也做完了。剩下的就是等结果了，希望能就此结束秋招吧，然后放松几天，去实习，开始新的生活

小米测评

  测评难得雅痞，之前也做过但不知道正确答案，就没怎么刷过这种测评，图形题和计算题看的好烦，不懂这有啥意义啊，难道只凭这种测评就能看出人是啥心理，啥智商了吗。要是是因为测评挂了，我直接原地裂开┭┮﹏┭┮



2020.11.13

  最近刷题感觉陷入了一个怪圈，只有简单题才能做，一遇到中等题或者困难题就想着看题解，这种思维感觉不太好，下次遇到中难题至少要想10分钟不会再看了。

  下午被辅导员要求去参加了学校另一个校区的线下招聘，都是些不想去的，还硬是催了我二十分钟，想了想就去看看，毕竟也没有线下面试过，体验一波流程吧。

  整个过程太尴尬了，三个企业宣讲。就第一个国家电网的子子公司有点排面，公司的大佬还有福利讲了挺多，后面两个小公司一个讲着没底气了，一个干脆不讲了，然后一结束没人投简历全部排着队出去了。。。合着全都是来凑数的。而且每个公司面试都只有HR，一般不是要有技术面试官的吗。。还是说我参加的少了，没见识。可惜省份有点尴尬，学校好像还是全省最好几所了，要是在北上广可能还能去蹭蹭其他学校大公司的招聘会

​

  秉持着鸡蛋不能放一个篮子的原则，又冲了几家补招，突然就不想实习了，反正也不call我，那就补招上岸把。还有最近看到的实习岗位好多都是go语言，感觉go语言好像真的是趋势了，谷歌影响力还是大啊，像腾讯这种龙头都在往go转了

  投递公司：

  • 快手
  • 达达
  • 顺丰

  今天又是堕落的一天，现在明明都没有offer，就开始休闲娱乐了，看了B站上一整页的马保国，真的鬼畜新星。然后晚上又开始贤者模式一般地醒悟过来，还是要去学习啊，要能应对各种突然面试才行，只要没有签，就不要停下来啊~



2020.11.16

  周末结束，又有很多新的补招信息，但是不知道为什么，看到了一些之前笔试面试失败的公司补招却不敢投，就有种底气不足的感觉，害怕又给虐一轮。转念一想，给虐那么多次了也不差这几次就狠下心投了

  • 京东广告部
  • 美团到店
  • 跟谁学
  • 途虎养车
  • 京东国际



2020.11.17

  好累啊，今天又刷了一天牛客，找补招信息，找来找去发现就那几家，太难了，但是补招这种情况感觉就真的很考验信息检索能力。之前投的其他补招都没有消息，搞得我现在又想去找实习了，没有竞争力真的太难了，然而现在实习到春节也不过三个月不到的时长，而且也没有啥好点的公司要，感觉只能去外包公司实习了，要是大三暑期实习感觉还能接受，可是现在已经大四了，接受不能啊。。。没顶住，投了个离学校比较近的实习

  • 金山


货拉拉二面

  这家的流程好慢啊，慢的我还以为之前一面已经挂了。

  因为没有特别想去，面试前毫无准备，甚至刷牛客，结果自然是面的不是很好，项目有些地方遗忘了，不过面试官的态度挺好的，但可能就是没有后面流程了吧。



2020.11.18

金山简历(淘汰)

  一早上起来就看到给挂了，感觉找个实习比校招还难，直接劝退。现在投实习可能真的晚了，还是专心复习准备春招补招吧

携程测评

  本来以为又要做一小时，结果发现和上次做的一样，北森测评只要一直点跳过就行了，不知道能不能给面试机会啊，上海是几个一线城市里最想去的了



2020.11.19

  莫名其妙地到了最后一面。。

货拉拉HR面

  感觉货拉拉的面试官都挺好的，就是我自己太搞了。。问我公司05分评积分，我说34分，现在感觉至少应该说个4分，有点太不给面子了。还问了给了offer是否会来，我觉得当时要是坚定一点可能就很稳吧，不过还是太实诚了，说不一定去，然而手上只不过有个泡池子的。

  面试前在了解公司情况的时候，看到这家公司的老板以前打德州扑克打七年成了财富自由，惊了！！


途虎养车笔试

  1. 两个有序数组，一个升序，一个降序，合并成升序数组
  2. 求排列组合公式C(r,n)，n! / r! / (n - r)!

  好久没做笔试，笔试题果然还是这么恶心人啊，第一题常规的双指针做法结果55%，第二题用动态规划求出所有阶乘然后计算12%，两个都是O(N)的时间复杂度然后都不能过，难道非得打表O(1)吗

  突然发现自己好像局限于牛客上的公司了，其实别的网站上也有一些公司在补招但没有信息出来，于是在拉勾网上又投了几家

  • 欢聚
  • 收钱吧
  • BIGO
  • Insta360
  • 字节教育



2020.11.20

货拉拉OC

  下午问了是否确定要offer，薪资比之前都要高，但还没想好要不要接受，给了三天时间考虑。不知道要不要放弃小米啊，但其实又不是特别想去北京，还是想留在南方。哎，人的欲望啊，无休无止，无穷无尽，感觉自己快要迷失了。

小米OC

  终于，一周时间，周末的前一天OC了，可能也要正式宣布我的秋招之旅结束了吧。今后要是再想冲击更好的平台可能就要等社招了吧，谁又能想到在11月中旬我这样的菜鸡也能一天双oc呢，周末休息两天，等正式接到offer签约后，再回来写完最后一段把



2020.11.21

小米offer

  接到了offer邮件。整个补招20天结束。说实话补招在难度上可能是几种招聘最小的了，没有笔试，岗位急缺，争的人也不多，被捞起来就有很大机会上岸，所以真的觉得自己很幸运了。虽然朋友们说应该再试试更好的，其实我也想，但是之前几个大厂面试的溃败让我看清自己目前的水平了，还是要继续沉淀。有时候感觉自信真的源于实力，因为准备不够充分，所以在面试大厂时会紧张，我想如果实力足够了应当就会从容应对吧。

  这段校招之旅真的改变了我很多，从原本只想着考试考好就行，到之后开始会为未来考虑，有主动提升自己的意识，懂得去获取资源，会开始尝试新的学习方式。

  对未来的希望：不断提升自己的技术，多看些技术书籍还有大牛的博客，增强自己的知识广度和深度，还有要多开阔自己的视野，不要让自己局限于学生思维， 也不知道自己在35岁了还会不会仍在这个行业里，但不管未来怎么样也只能不断摸索着前进，要敢于走出自己的舒适圈，能适应学习不同的技术栈(有时候真的觉得自己过于局限于Java这块了)，还有之后要开始每天锻炼身体，再也不熬夜了，毕竟在这个动荡的2020年能好好活着就很不容易了~~
